# DealSource.ai

Real estate deal analyzer — 6 strategies, instant calculations, Supabase backend.

## Deploy in 5 minutes

### 1. Upload to GitHub
1. Go to github.com → sign up → New repository → name it `dealsource-ai`
2. Upload ALL these files keeping the folder structure:
   ```
   package.json
   public/
     index.html
   src/
     index.js
     App.jsx
   ```

### 2. Deploy to Vercel
1. Go to vercel.com → Sign up with GitHub
2. Click "Add New Project" → Import `dealsource-ai`
3. Leave all settings as default → click **Deploy**
4. You'll get a live URL like `https://dealsource-ai.vercel.app`

### 3. Add your Vercel URL to Supabase
1. Go to Supabase → Authentication → URL Configuration
2. Set **Site URL** to your Vercel URL
3. Add it to **Redirect URLs** too

That's it — your app is live!
